# ⚽ Gestión de Turnos para Canchas de Fútbol - Backend

API REST desarrollada con ASP.NET Core para la gestión de reservas de canchas de fútbol.

## 🚀 Tecnologías utilizadas

- ASP.NET Core
- C#
- Entity Framework Core
- SQL Server
- Swagger

## ✨ Funcionalidades

- Registro de usuarios.
- Inicio de sesión.
- Gestión de canchas.
- Gestión de turnos.
- Validación de reservas.
- API REST.
- Documentación con Swagger.

## 📁 Instalación

Clonar el repositorio

```bash
git clone https://github.com/Mariano-404/TU-REPOSITORIO.git
```

Ingresar a la carpeta

```bash
cd API
```

Restaurar dependencias

```bash
dotnet restore
```

Ejecutar la API

```bash
dotnet run
```

## 📖 Swagger

Una vez iniciada la API:

```
https://localhost:xxxx/swagger
```

## 🔗 Frontend

Esta API es consumida por una aplicación desarrollada en Angular.
